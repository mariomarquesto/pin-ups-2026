Pin Ups Store 2026
Bienvenido al repositorio oficial de Pin Ups Store, un ecommerce especializado en moda inclusiva, talles reales y calzado con diseño especial, basado en Tucumán, Argentina.

🚀 Descripción del Proyecto
Este proyecto es una Single Page Application (SPA) desarrollada con React que busca ofrecer una experiencia de compra fluida, enfocada en la comodidad del usuario y la diversidad corporal. El sitio cuenta con animaciones personalizadas, diseño responsive y una interfaz intuitiva para la gestión de productos y atención al cliente.

🛠 Tecnologías Utilizadas
Frontend: React.js

Estilos: React-Bootstrap & CSS3 (Custom animations)

Iconografía: React-Icons

Enrutamiento: React-Router-DOM

Despliegue/Entorno: Node.js, Git/GitHub

📦 Características Principales
Diseño Responsive: Adaptado para móviles, tablets y escritorio.

Experiencia de Usuario: Animaciones de entrada (reveal effect) al hacer scroll.

Catálogo Inclusivo: Enfoque en talles grandes (1X a 5XL) y calzado ancho (EE/EEE).

Integración: Enlaces directos a WhatsApp para atención personalizada y gestión de pedidos.

Sección "Nosotras": Historia de la marca y valores centrados en el amor propio y la diversidad.
